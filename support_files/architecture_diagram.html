💊 CliniMatch — Architecture & Tech Stack

AI-Powered Clinical Trial Matching System · Group 13 · Milestone 1

System Architecture — 3-Layer Pipeline
🖥️ Application Layer
FRONTEND
User-facing Streamlit dashboard — input, visualization, interaction
Patient Input Form
STREAMLIT
Sidebar form: age, gender, diagnosis, treatments, medications, location. Load sample profiles or enter manually.
Ranked Trial Cards
STREAMLIT
Cards with match scores, badges, phase, sponsor. Expandable details for eligibility, evidence, locations.
Weight Sliders
STREAMLIT WIDGETS
5 adjustable sliders for ranking priorities: eligibility, distance, phase, urgency, evidence.
Evidence Panel
STREAMLIT TABS
Tabbed views: score breakdown, eligibility criteria (pass/fail), Semantic Scholar + PubMed evidence, trial locations with map.
🧠 Intelligence Layer
AI / ML
Core reasoning — disease extraction, eligibility evaluation, evidence verification, ranking
Disease Extraction
GPT-4.1 MINI
Extracts structured disease info from free-text patient profile. Outputs condition name, keywords, stage, key factors.
Eligibility Filter
GPT-4.1 MINI
Evaluates each trial's inclusion/exclusion criteria against patient. Understands medical synonyms (semantic matching).
Evidence Verification
SEMANTIC SCHOLAR + BIOPYTHON
Checks drug efficacy via Semantic Scholar (with citation counts) and PubMed (peer-reviewed papers). Cross-references both sources. Both FREE.
Multi-Objective Ranker
PYTHON + GEOPY
Scores trials across 5 dimensions (eligibility, distance, phase, urgency, evidence). Normalizes to 0-1, applies user weights.
🗄️ Data & Infrastructure Layer
DATA SOURCES
External APIs and data sources feeding the pipeline
ClinicalTrials.gov API v2
REST API · FREE
500K+ trials. Returns JSON: eligibility criteria, phases, enrollment, site locations (lat/lng), sponsors, dates.

REST API · FREE
Academic paper search with citation counts, abstracts, DOIs. Drug efficacy evidence from peer-reviewed research.
PubMed / NCBI
ENTREZ API · FREE
30M+ peer-reviewed papers. Search by drug + condition. Returns titles, abstracts, DOIs.
Synthetic Patients
JSON FILES · INTERNAL
3-5 hand-crafted HIPAA-safe test profiles. Breast cancer, renal cell carcinoma, lung cancer.

Key Differentiators vs Existing Tools
Semantic Eligibility Matching
Unlike keyword search (ClinicalTrials.gov), our LLM understands medical synonyms: "high BP" = "hypertension", "tumor" = "neoplasm". Evaluates complex nested logic (AND/OR/UNLESS conditions).
Evidence-Based Verification
No other free tool automatically cross-references drug evidence from Semantic Scholar + PubMed. We don't just match — we verify the treatment is actually promising.
Multi-Objective Ranking
5-dimensional scoring with user-adjustable weights. Not just "match/no match" — we rank by eligibility, distance, phase, urgency, and evidence strength simultaneously.
